Sistema de Gestión de Biblioteca Universitaria

Universidad Andina del Cusco - Facultad de Ingeniería y Arquitectura

Introducción al Proyecto

Implementación de 10 funciones UDFs para la gestión eficiente de una biblioteca universitaria, incluyendo 5 funciones escalares y 5 funciones de tabla que automatizan procesos críticos del sistema.

Objetivos

  • Automatizar cálculo de multas
  • Validar formatos de ISBN
  • Generar códigos únicos
  • Generar reportes estadísticos

Beneficios

  • Reutilización de código
  • Centralización de lógica
  • Consistencia en cálculos
  • Optimización de rendimiento

Funciones Escalares (5)

CalcularMultaRetraso()
Escalar
Calcula multas por retraso en devolución de libros considerando días de gracia y tope máximo.
Parámetros: @FechaDevolucionEsperada, @FechaDevolucionReal, @MultaPorDia, @DiasGracia
Retorno: DECIMAL(10,2)
ValidarISBN()
Escalar
Valida formato ISBN-10 e ISBN-13 usando algoritmos de verificación oficiales.
Parámetros: @ISBN VARCHAR(20)
Retorno: BIT (1=válido, 0=inválido)
FormatearNombreCompleto()
Escalar
Formatea nombres de usuarios con títulos según tipo (Dr., Est., Lic.) en estilo "APELLIDO, Nombre".
Parámetros: @Nombre, @Apellido, @TipoUsuario
Retorno: VARCHAR(150)
GenerarCodigoLibro()
Escalar
Genera código único interno para libros combinando categoría, año y últimos dígitos del ISBN.
Parámetros: @ISBN, @AnioPublicacion, @Categoria
Retorno: VARCHAR(50)
CalcularIndiceUsuario()
Escalar
Calcula índice de responsabilidad del usuario basado en préstamos vencidos vs totales.
Parámetros: @UsuarioID
Retorno: DECIMAL(5,2) (0-100)

Funciones de Tabla (5)

fn_ObtenerLibrosPorCategoria()
Tabla
Retorna todos los libros filtrados por categoría específica con información detallada.
Parámetro: @CategoriaNombre (opcional)
Retorno: TABLE
fn_ObtenerPrestamosActivosDetalle()
Tabla
Muestra préstamos activos o vencidos con información detallada de usuarios y libros.
Parámetro: @UsuarioID (opcional)
Retorno: TABLE
fn_EstadisticasUsuario()
Tabla
Genera reporte completo de estadísticas del usuario incluyendo índice de responsabilidad.
Parámetro: @UsuarioID
Retorno: @Estadisticas TABLE
fn_LibrosDisponibles()
Tabla
Lista libros disponibles filtrados por categoría y/o editorial con estado de disponibilidad.
Parámetros: @Categoria, @Editorial (opcionales)
Retorno: TABLE
fn_HistorialCompletoUsuario()
Tabla
Combina historial de préstamos con actividad actual para auditoría completa.
Parámetro: @UsuarioID
Retorno: @Historial TABLE

Flujo del Sistema Integrado

1

Validación de Libro

ISBN → ValidarISBN() → [Válido] → GenerarCodigoLibro()

2

Gestión de Préstamo

Solicitud → fn_EstadisticasUsuario() → Verificación índice → Aprobación

3

Control de Devolución

Devolución → CalcularMultaRetraso() → Actualización multas → CalcularIndiceUsuario()

4

Reportes y Auditoría

fn_HistorialCompletoUsuario() + fn_EstadisticasUsuario() → Reportes consolidados
